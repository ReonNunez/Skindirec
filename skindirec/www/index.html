About the Application

Skin DiRec is an application developed by Electronics Engineering students from De La Salle Lipa. It uses image processing and deep transfer learning in identifying common skin diseases. The following are the skin diseases that it can identify:
1. Atopic Dermatitis
2. Contact Dermatitis
3. Nummular/ Discoid Eczema
4. Psoriasis
5. Ringworm
Disclaimer: This application does not give final diagnosis. It is only for the identification of the stated skin diseases.

Instructions

1. Select the camera button shown in the Homepage.
2. Take a picture or import an image from your device.
3. You will have the option to approve the captured image or retake another one.
4. Wait for a few seconds while the application processes your image for classification.
5. Once finished, the results will be displayed on your screen.
6. View the first aid advice and when-to-see a doctor advice to help you with your concern.

Note:
1. The application only currently supports .jpg and .jpeg files for upload.
2. This application uses cordova-android version 9, which only supports Android versions 5.1 and above.
